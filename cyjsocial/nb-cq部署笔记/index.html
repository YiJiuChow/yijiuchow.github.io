nb-cq部署笔记

python nonebot cq-http
發布日期:   2021-12-07

Linux

ubuntu 20.04

一. 做一些必要准备

1.更新 apt

sudo apt update

2.更新 pip

python3 -m pip install –upgrade pip

3.安装 nb-cli 和 nonebot2

pip install nb-cli

pip install nonebot2

二. 创建一个nb

1.根据自己需求选择(这里chow替换成自己想要设置的文件名)

nb create chow

2.进入nb文件夹

cd chow

3.启动bot

nb run 或者 python3 bot.py

默认端口是8080(可以使用vim .env.dev更改,同时可以配置一些环境变量,设置超级用户等,具体见nonebot2文档)

4.安装插件,这里建议使用在bot目录直接使用nb命令,当然用pip也可以,只是需要在bot.py中添加相应插件

nb plugin install 插件名
