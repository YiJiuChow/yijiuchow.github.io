3).chapter03 变量

变量定义 variable

  1. 变量是程序的基本组成单位
  2. 变量相当于内存中的一个数据存储空间的表示，你可以把变量看做是一个房间的们牌号，通过门牌号我们可以找到房间，而通过变量名可以访问到变量
  3. 变量使用的基本步骤 声明->赋值
  4. 不同变量，类型不同，占用的空间大小不同
  5. 变量在同一作用域不能重名

基本数据类型 basic data type

  1. 每一种数据都定义了明确的数据类型，在内存种分配了不同大小的内存空间（字节）
基本数据类型 引用数据类型
数值型{整型（byte short int long） 浮点型（float double）} 类class
字符型char 2字节 接口interface
布尔型boolean 1字节 数组[]
类型 占用存储空间 范围
byte字节 1字节 -128~127
short短整型 2字节 -32768~32767（2的15次方）
int整型 4字节 -2147483648~2147483647（2的31次方）
long 长整型 8字节 -2的63次方~2的63次方-1
float单精度 4字节 -3.403E38~3.403E38
double双精度 8字节 -1.798E308~1.798E308
char 字符 2字节 0~65535
  1. 1byte = 8bit bit：最小存储单位，byte：基本存储单位，数值型的第一个bit表示正负
  2. 关于浮点数在机器中存放形式的简单说明，浮点数=符号位+指数位+尾数位 ，尾数部分可能丢失，造成精度损失（小数都是近似值）
  3. char本质是一个整数，默认输出时，unicode码对应的字符

简单了解字符编码

  1. ASCII(一个字节表示，128个字符，实际上1个字节可以表示256个字符，只用了128个)
  2. Unicode(固定大小的编码，使用2个字节来表示字符，字母和汉字统一都是占用2个字节，这样浪费空间),兼容ASCII编码
  3. UTF-8(大小可变的编码，字母使用1个字节，汉字使用3个字节)，是Unicode编码的改进版
  4. GBK(可以表示汉字，而且范围广，字母使用1个字节，汉字使用2个字节)
  5. gb2312(可以表示汉字，gb2312 < GBK)
  6. big5码(繁体中文，台湾，香港)

基本数据类型转换

1.自动类型转换(auto-convert)
  1. 当Java程序在进行赋值或者运算时，精度小的类型自动转换为精度大的数据类型
  2. 数据类型按精度(容量)大小排序

char -> int -> long -> float -> double

byte -> short -> int -> long -> float ->double

  1. 当有多种类型的数据混合运算时，系统首先自动将所有数据转换成容量最大的数据类型，然后再进行计算
  2. char，byte，short 之间不发生自动类型转换，当把具体的数值赋给它们时，先判断该数是否在其范围内,如果使用变量赋值，则判断类型
  3. char，byte，short 在计算时，首先转换成int
  4. boolean 不参与自动转换
  5. 自动提升原则：表达式结果类型自动提升为操作数中最大的类型
2.强制类型转换(force-convert)
  1. 当进行数据从大到小转换时，就需要使用强制类型转换
  2. 需要转换一串运算结果时，常用小括号提升优先级
  3. 强制类型转换会造成精度损失
